Number Guessing Game - Assignment 4

Description:
This is a simple Number Guessing Game built in Python. The program randomly selects a number between 1 and 100, and the player has 10 attempts to guess it. After each guess, the game provides feedback, telling the player if their guess was too high or too low.

How to Run:

1. Using Python directly:
   - Ensure Python 3 is installed.
   - Run the command: `python app.py`

2. Using Docker:
   - Ensure Docker is installed and running.
   - Run the command: `docker run -it jonathanjohnn/number-guessing-game:v1.0`
